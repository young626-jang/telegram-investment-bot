# 📱 Sofia Investment Assistant

실시간 투자 뉴스 알림 시스템

## 🚀 기능

- **실시간 뉴스 모니터링**: IBM, ServiceNow, SoundHound AI 종목
- **스마트 알림**: 중요 뉴스 발생 시 텔레그램 즉시 알림
- **포트폴리오 분석**: 현재 손익 상황 실시간 계산
- **매일 아침 브리핑**: 한국시간 오전 9시 자동 전송
